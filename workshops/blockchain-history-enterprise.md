# Part 1: The Genesis
## 2008-2013

---

# October 31, 2008
## The Bitcoin Whitepaper

- Satoshi Nakamoto publishes "Bitcoin: A Peer-to-Peer Electronic Cash System"
- **9 pages** that changed everything
- Key innovation: **Proof of Work** consensus
- First solution to the "double spend" problem without a central authority

---

# January 3, 2009
## Block 0 — The Genesis Block

> "The Times 03/Jan/2009 Chancellor on brink of second bailout for banks"

- First Bitcoin mined
- Message embedded in the block — a timestamp and a statement
- **The first immutable record on a public blockchain**

# Public vs. Private Blockchains

**Public (Bitcoin, Ethereum)**
- Anyone can participate
- Maximum decentralization
- Most secure, most immutable

**Private (Hyperledger, R3)**
- Permissioned access
- Faster, more controlled
- Less censorship-resistant

**Hybrid:** Use public chains for anchoring, private systems for data

---

# What is "Anchoring"?

1. Take any document or data
2. Create a cryptographic fingerprint (hash)
3. Write that fingerprint to a blockchain
4. **Result:** Tamper-evident timestamp

The document never touches the blockchain — only the proof does.

---

# Why Bitcoin for Enterprise Anchoring?

- **17 years** of continuous operation
- **Never hacked** at protocol level
- **$1T+** securing the network
- **Most decentralized** — no single point of failure
- **Most legally recognized** timestamp source

# Appendix: Glossary

- **Hash:** Cryptographic fingerprint of data
- **Block:** Bundle of transactions/data
- **Node:** Computer running blockchain software
- **Consensus:** Agreement mechanism (PoW, PoS)
- **Smart Contract:** Self-executing code on blockchain
- **Anchoring:** Writing proof to blockchain without storing data
